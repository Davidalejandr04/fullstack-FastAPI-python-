# Login y gestión de tareas

En este proyecto presento la construcción de un backend utilizando el lenguaje de programación Python con fastAPI  y un frontend desarrollado con React (Vite).
La aplicación permite a los usuarios registrarse, iniciar sesión mediante autenticación JWT y gestionar sus tareas personales de manera segura.

## Tecnologías utilizadas

### Backend
- Python
- FastAPI
- JWT Authentication
- SQLAlchemy
- Docker

### Frontend
- React
- Vite
- React Router
- Fetch API
- Javascript

This is a full-stack task management application built with FastAPI (backend) and React + Vite (frontend). 

The application implements JWT-based authentication and protected routes, allowing users to securely manage their personal tasks.
